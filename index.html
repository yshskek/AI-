프로젝트: AI 동화책 작가

(부제: 나만의 이야기 생성기)

[문제 정의]

(예시: 기존의 동화책은 이미 정해진 이야기이며, 아이들은 수동적인 독자에 머무릅니다. 부모들은 아이에게 교훈적이면서도 새로운 이야기를 들려주고 싶어 하지만 매번 창작하는 것은 어렵습니다. 핵심 목표는, 아이가 직접 주인공, 배경, 사건 등 간단한 키워드만 입력하면, AI가 즉석에서 아이의 눈높이에 맞는 고유한 삽화가 포함된 동화책을 생성해 주는 웹 애플리케이션을 개발하는 것이었습니다.)

[아키텍처/설계]

(예시: 순수 HTML/CSS/JavaScript로 사용자가 키워드를 입력할 수 있는 UI를 구축했습니다. 사용자가 '이야기 만들기'를 요청하면, 로컬에서 실행 중인 Python `FastAPI` 서버(`http://127.0.0.1:8000`)에 API 요청을 보냅니다. FastAPI 백엔드는 이 요청을 받아, 로컬 `Ollama`를 호출하여 동화 텍스트를 생성(1차)하고, 이 텍스트로 삽화 프롬프트를 생성(2차)합니다. 마지막으로 OpenAI `DALL-E 3` API를 호출(3차)하여 삽화를 생성하고, 모든 데이터를 프론트엔드로 반환합니다.)

[핵심 기능 1: AI 기반 맞춤형 동화 텍스트 생성]

(예시: 사용자가 주인공, 배경, 사건을 입력하면, 이를 변수로 하는 시스템 프롬프트를 구성합니다. "당신은 5살에서 8살 사이의..."로 시작하는 이 프롬프트는 로컬에 설치된 `Ollama`의 `anpigon/eeve-korean-10.8b` 모델에 전달되어, 입력된 키워드가 자연스럽게 녹아든 순수 한글 동화 텍스트를 생성하도록 설계했습니다. 폭력성이나 부정적인 콘텐츠를 배제하는 가드레일을 프롬프트에 포함했습니다.)

[핵심 기능 2: AI를 통한 동화 삽화 자동 생성]

(예시: 텍스트가 성공적으로 생성되면, 1차 생성된 동화 텍스트를 기반으로 2차로 `Ollama`를 다시 호출하여 삽화용 프롬프트를 생성합니다. 이 한국어 프롬프트를 `OpenAI DALL-E 3` API에 요청하여 1024x1024 크기의 고품질 삽화를 반환받습니다. 이 이미지는 Base64 형식으로 프론트엔드에 전달되어 텍스트와 함께 배치, '그림책' 경험을 구현했습니다.)
